## Login

Anda bisa memulai aplikasi porta pro web dengan login terlebih dahulu:

1. Akses url https://apps.portapro.co.id/ pada perangkat anda, lalu akan tampil halaman login seperti berikut:

<Image
    loading="lazy"
    src={screenLogin}
    width={1363}
    height={544}
    style={{ margin: "auto" }}
></Image>

2. Isi kode perusahaan anda yang sudah terdaftar pada aplikasi Porta Pro
3. Isi username dan password dengan data user yang telah telah terdaftar pada aplikasi Porta Pro/ECes.Q
