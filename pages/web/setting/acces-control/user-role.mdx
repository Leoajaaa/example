## Hapus Data

<Steps>

### Pilih User Role

Tentukan data yang akan dihapus, kemudian Klik 1x pada data tersebut yang
tampil di tabel.


### Non aktifkan User Role

1. Tentukan data yang akan diedit, kemudian Klik 1x pada data tersebut yang tampil di tabel.
2. Sistem akan menampilkan data User Role, pilih status menjadi non aktif
3. Klik tombol **Simpan** untuk menyimpan data.



<Callout type="error" emoji="️🚫">
  `Penonaktifan User Role data hanya bisa dilakukan jika User Role tersebut belum terhubung ke data pada modul lain, seperti : stok unit User Role, penjualan serta modul-modul lainnya`
</Callout>

</Steps>
