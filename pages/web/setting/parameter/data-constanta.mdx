# Data constanta

## Tambah Data Constanta

<Steps>
### Klik **Buat Baru**

### Info Constanta

<Image
            src={constantaa1}
            
            width={1000}
            height={1000}
                
        />

1. Isikan pada text box **Kode Constanta**,**Nama Constanta**, **Keterangan**, **Nama Label**, **Nilai Constanta 1** dan **Nilai Constanta 2**.
2. Pilih Nama Constanta
3. Pilih Kode Akun

<Callout type="info" emoji="ℹ️">
    `Tanda (*) pada atribut isian, menandakan bahwa isian tersebut wajib diisi/tidak boleh kosong`
</Callout>

### Klik Simpan

Klik tombol **Simpan** untuk menyimpan Data Constanta
</Steps>
