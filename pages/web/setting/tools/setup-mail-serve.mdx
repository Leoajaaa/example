# Setup Mail Server

## Tambah Setup Mail Server

<Steps>
### Klik **Buat Baru**

### Info Mail Server

<Image
            src={maila1}
            
            width={1000}
            height={1000}
                
        />

1. pilih proyek
2. Isikan pada text box **SSL Addr**,**SSL Port**, **From Mail**, **From Name**, **Password**.
