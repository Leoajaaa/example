# Proses KPR

Menu **Proses KPR** berfungsi untuk input data status proses KPR yang diajukan oleh
konsumen terkait pembelian unit dengan cara pembayaran KPR.

## Langkah-langkah menambahkan data proses KPR

    <Steps>
    ### Pilih nama **Proyek**.
    ### Pilih nama **Bank** pemberi KPR.
    ### Klik tombol **Tampilkan** untuk mulai pencarian data.
    ### Ubah data di tabel
    <Image
            src={proseskpru1}
            
            width={1000}
            height={1000}
                
        />
    1. Berikan tanda Checlist di kolom **PILIH** untuk baris data yang akan diinput.
    2. Pilih Bank yang diinput
    3. Isikan tanggal, Ceklist kolom sudah (Jika status proses sudah selesai), pilih hasil proses (Disetujui/Ditolak/-) dan isi keterangan
    ### Klik tombol **Simpan** untuk menyimpan data
    </Steps>
