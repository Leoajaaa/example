# <span style={{ fontSize: '28px' }}> Jurnal Umum </span>

Menu Jurnal Umum, digunakan untuk membuat jurnal secara manual seperti Jurnal Memorial, Amortisasi dan Jurnal Penyesuaian.

1. pilih proyek
2. lengkapi isian pada jurnal umum tanggal jurnal, nomor jurnal, tanggal bukti,

3. lengkapi tabel dengan nama akun, sub akun, dengan menggunakan tombol cari,

## <span style={{ fontSize: '24px' }}> Edit Data </span>

<Steps>

### pilih proyek

pilih proyek (contoh : MSI land) tentukan data yang akan diedit, kemudian klik 1x pada data tersebut yang tampil di
tabel.

### Isi perubahan data

<Image
            src={jue1}
            
            width={1000}
            height={1000}
                
        />

System akan menampilkan data jurnal umum (pilih proyek, tanggal jurnal, nomor
jurnal, tanggal bukti, nomor bukti, tanggal referensi, nomor referensi dan keterangan).

1. Isikan pada tabel kode akun.
2. (optional) pilih nama akun , dengan cara klik tombol cari sistm akan
menampilkan UI list akun.
3. (Optional) Pilih sub akun, dengan cara klik tombol cari system akan menampilkan
UI list sub akun.
4. Isikan debet dan kredit menggunakan angka
5. (Optional) pilih rekanan dengan cara klik tombol cari system akan menampilkan
UI list Rekanan.
6. Isikan uraian

### Klik Update

7. Klik **update** jika ingin mengupdate data jurnal umum
8. Kemudian muncul popup konfirmasi klik update jika ingin mengupdate dan batal
jika batal untuk mengupdate
9. Kemudian muncul popup pemberitahuan bahwa data berhasil di update, jika ingin
datanya diprnit out maka klik print out traksasi, jika tidak maka klik tombol
selesai.

</Steps>
