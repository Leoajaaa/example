# Penerimaan Kas-Bank

Menu **Penerimaan Kas-Bank**, berfungsi untuk memfasilitasi perpindahan sejumlah uang dari/ke Kas - Setara Kas, (Kas ke Kas atau Kas ke Bank 
atau Bank ke Bank). Transaksi ini hanya berlaku untuk Rekening/Kas Proyek yang sama.

## Input Data Penerimaan Kas-Bank

<Steps>
### Klik buat baru
Tombol aksi buat baru terdapat pada halaman awal modul Pindah.
<Image
            src={penerimaana1}
            
            width={1000}
            height={1000}
                
        />
### Pilih proyek
Lengkapi isian pada atribut tanggal penerimaan, nomor bukti penerimaan, diterima
dari,tanggal referensi, nomor referensi, cara bayar, no rekening,kas/bank, referensi :
no cek/giro/edc,
### Diterima dari menggunakan icon cari
system akan menampilkan popup list rekanan
1. Pilih rekanan (rekanan, konsumen, karyawan, bank, dan rekening)
2. Pilih cara bayar (cash, giro/cek, transfer,dan EDC)
3 Pilih no.rekening menggunakan icon cari
System akan menampilkan list rekening
### Isi tabel akun
Lengkapi isian (kode akun, akun,subakun, unit property, nominal(RP),uraian)
1. Klik pada icon PPN dan akan otomatis terisi
2. Isi biaya lainnya
3. Klik **simpan** untuk menyimpan klik **batal** untuk membatalkan simpan
</Steps>
