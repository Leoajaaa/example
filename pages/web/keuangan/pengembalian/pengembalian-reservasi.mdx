# Pengebalian Reserve

Menu **Pengebalian Reserve**, digunakan untuk memfasilitasi
Pengembalian uang dari Reservasi penjualan ke Konsumen.

## Langkah-langkah menambahkan Pengebalian Reserve

    <Steps>
    ### Klik tombol **Buat Baru**.
    ### Isikan data pada halaman tambah data.

    1. Pilih nama **Proyek**
    2. Pilih **tanggal Pengembalian**

    `Catatan: nomor bukti akan terisi otomatis berdasarkan tanggal bukti, proyek dan rekening`

    <Tabs
            items={[
                "Data Penjualan",
                "Rekening Pengeluaran",
                "Detail Pengeluaran",
            ]}
        >
            <Tab>
        <Image
            src={pra1}
            
            width={1000}
            height={1000}
                
        />

            3. Pilih Nomor Reservasi, dengan cara menekan tombol <Image src={search} alt="search akun" width={"auto"} height={"auto"} style={{ display: "inline-block" }}/>, Sistem akan
                menampilkan UI List Reservasi Unit. Pilih Nomor Reff yang tertera pada table List Reservasi.
                Sistem akan otomatis menampilkan Informasi Proyek, Nama Customer, Unit Properti dan Nominal yang diterima sebelumnya dari konsumen berdasarkan penerimaan yang tercatat pada sistem.
          
            </Tab>

            <Tab>

         <Image
            src={pra2}
            
            width={1000}
            height={1000}
                
        />

            Lengkapi isian data atribut (metode pembayaran,no.rekening,bank,referensi no.cek/giro/EDC)

            4. Pilih metode pembayaran (Giro/Cek,cash,tranfer,dan EDC)

            5. Pilih no.rekening dengan klik icon cari, bank akan otomatis terisi Ketika sudah mengisi no.rekening

            6. Masukan referensi no.Cek/Giro/EDC
            
            </Tab>
            <Tab>
        <Image
            src={pra3}
            
            width={1000}
            height={1000}
                
        />
            `Catatan: Total penerimaan akan terisi otomatis berdasarkan nilai penerimaan yang sudah di inputkan`
            
            7. lengkapi data pada text box di tab detail pengeluaran (nilai pengembalian, Keterangan)
            </Tab>
    </Tabs>
    ### Klik tombol **Simpan** untuk menyimpan data.
    Klik tombol **Print Out Pengebalian Reserve** lalu
    Klik tombol **Kwitansi** apabila akan mencetak kwitansinya, Klik tombol **Print Bukti Pembayaran** apabila akan mencetak bukti pembayaran.

    </Steps>
