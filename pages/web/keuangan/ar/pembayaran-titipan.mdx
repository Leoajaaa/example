# Pembayaran Titipan

Menu **Pembayaran Titipan** ini digunakan untuk mencatat penerimaan uang titipan dari konsumen
yang sumbernya berasal dari item nilai titipan pada booking/ kontrak penjualan
(Biaya Notaris, BPHTB, Admin Bank, dll).

## Langkah-langkah menambahkan pembayaran titipan

<Steps>
    1. Pilih nama **Proyek**.
    2. Pilih **tanggal bukti**.

    `Catatan: nomor bukti akan terisi otomatis berdasarkan tanggal bukti, proyek dan rekening`

    <Tabs
            items={[
                "Data Penerimaan",
                "Detail Penerimaan",
            ]}
        >
            <Tab>
            <Image
            src={pta1}
            
            width={1000}
            height={1000}
                
        />
            3. Pilih **Nomor Kontrak**, dengan cara menekan tombol <Image src={search} alt="search akun" width={"auto"} height={"auto"} style={{ display: "inline-block" }}/>, Sistem akan
            menampilkan UI List Kontrak. Pilih Nomor Unit yang tertera pada table List Kontrak,
            sistem akan otomatis menampilkan Informasi Proyek, Nama Customer dan Unit Properti.

            `Catatan: pastikan tunggakan pembayaran titipan ditemukan di sistem.`

            4. Checklist **jenis penerimaan**.
            5. Tentukan **metode pembayaran**.
            6. Tentukan Kas-Bank (Rekening) untuk menempatkan dana yang diterima,
            dengan cara tekan tombol <Image src={search} alt="search akun" width={"auto"} height={"auto"} style={{ display: "inline-block" }}/>. Sistem akan menampilkan Daftar Rekening
            pada tabel UI List Rekening. Pilih nama kas/bank pada table.
            7. Isi nomor referensi/ giro/ cek dan nomor kwitansi(jika dibutuhkan).
            8. Isikan keterangan.

            </Tab>

            <Tab>
            <Image
            src={pta2}
            
            width={1000}
            height={1000}
                
        />
            9. Isi nilai pembayaran.
            `Catatan: Pastikan Nominal pembayaran tidak bisa lebih besar dari saldo titipan`

            10. Isikan keterangan.

            </Tab>
    </Tabs>
    ### Klik tombol **Simpan** untuk menyimpan data.
    Klik tombol **Kwitansi** apabila akan mencetak kwitansinya, Klik tombol **Print Penerimaan** apabila akan mencetak bukti penerimaan.

</Steps>
