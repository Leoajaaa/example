# <span style={{ fontSize: '28px' }}> Aktivitas Proyek </span>

Menu Aktivitas Proyek digunakan untuk memonitor semua proses aktivitas proyek dan bisa menampilkan data gambar aktivitas proyeknya sudah sejauh mana, serta menampilkan nilai presentase progres aktivitasnya. 

## <span style={{ fontSize: '24px' }}> Langkah-langkah menampilkan Aktivitas Proyek </span>

<Steps>
### <span style={{ fontSize: '22px' }}> Pilih Proyek </span>
Pilih proyek untuk menentukan aktivitas proyek mana yang akan ditampilkan (contoh: MSI Land1)

## <span style={{ fontSize: '24px' }}> Langkah-langkah update Aktivitas Proyek </span>

<Steps>

### <span style={{ fontSize: '22px' }}> Klik tombol update </span>
pilih data pada tabel mana yang ingin diupdate, kemudian klik tombol update dan sistem menampilkan popup Update Ativitas Proyek
<Image
            src={aktivitasu1}
            
            width={500}
            height={500}
                
        />
        1. Untuk tanggal update akan akan otomatis terisi
2. Pilih status (Belum dimulai, Proses, dan Selesai)
3. Seret bar progress saat ini untuk menentukan nilai persen progress aktivitas
proyek

4. Unggah file dalam bentuk (JPEG,JPG, &amp; PNG) maksimal lebar 2000px
5. Masukan Keterangan
6. Klik update
System akan menampilkan popup konfirmasi update proses aktivitas proyek
kemudian klik “iya, Update” untuk update dan “tidak” untuk tidak update
</Steps>
