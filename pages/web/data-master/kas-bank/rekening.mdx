<Tabs items={["Rekening", "Kas/Bank", "Bank Eksternal"]}>
    <Tab>
    Menu **Rekening** digunakan untuk mengidentifikasi rekening yang akan
    dilibatkan pada saat proses transaksi, baik rekening Kas ataupun Bank.

     ## Langkah langkah membuat rekening
        <Steps>
        ### Klik tombol **Buat Baru**

        ### Isikan kolom tambah data rekening
        <Image
            src={rekeninga1}
            
            width={1000}
            height={1000}
                
        />
        1. Pilih nama **Proyek**.
        2. Pilih nama kas/bank, dengan cara klik tombol dengan cara klik tombol <Image src={search} alt="search akun" width={"auto"} height={"auto"} style={{ display: "inline-block" }}/>
        sistem akan menampilkan UI kas/bank.
        3. Isikan pada text box kode rekening dan atas nama.
        4. Pilih salah satu kategori rekening.
        5. Pilih jenis rekening.
        6. Pilih nama akun, dengan cara klik tombol dengan cara klik tombol <Image src={search} alt="search akun" width={"auto"} height={"auto"} style={{ display: "inline-block" }}/>
        sistem akan menampilkan UI akun.
        7. Pilih status rekening.

        ### Klik tombol **Simpan** untuk menyimpan data
        </Steps>

     ## Langkah langkah mengubah rekening

        3. Ubah pada text box kode rekening dan atas nama.
        4. Ubah salah satu kategori rekening.
        5. Ubah jenis rekening.
        6. Ubah nama akun, dengan cara klik tombol dengan cara klik tombol <Image src={search} alt="search akun" width={"auto"} height={"auto"} style={{ display: "inline-block" }}/>

        7. Ubah status rekening.
        ### Klik tombol **Update** untuk mengubah data.

        <Steps>
        ### Klik No. Rekening pada data yang akan dihapus, yang tampil di tabel.
        ### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.
        ### Klik tombol **Hapus** untuk menghapus data.
        Klik **Hapus** untuk menghapus data, Klik “Cancel” untuk membatalkan

      ## Langkah langkah membuat bank eksternal
       <Steps>
          ### Klik tombol **Buat Baru**

          ### Isikan kolom tambah data bank eksternal
          <Image
            src={banka1}

            height={1000}
                
        />
          1. Isikan pada text box nama, kode kriling, kode RTGS dan status.

          ### Upload Logo
          Upload logo digunakan untuk melakukan upload format template file
          cetak per proyek. Drag gambar yang sudah ditentukan atau Klik tombol “Unggah Gambar” Maka akan muncul tampilan lokasi yang inging diupload.
          `Catatan: Format upload harus JPG/JPEG/PNG dan ukuran upload tidak lebih dari 2MB.`
