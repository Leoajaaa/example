# Kategori Unit

Menu **Kategori Unit** digunakan untuk mengelompokkan jenis properti,
contoh (rumah, ruko, apartemen, dsb).

## Langkah-langkah menambah kategori unit

<Steps>

### Klik button **Buat Baru**

### Isi pada Text Box kode kategori dan nama kategori

### Pilih status kategori

### Klik tombol **Simpan**, untuk menyimpan.

`Catatan: kode, nama dan status kategori wajib diisi.`

</Steps>

## Merubah kategori unit melalui tabel

Anda bisa merubah data status kategori unit di tabel dengan mengklik salah satu status yang diinginkan di dalam kolom checkbox.

![ubah kategori properti 1](@/public/image/web/data-master/properti/kategori-unit/kategori_unit1.png)

<Steps>

</Steps>

## Langkah-langkah merubah kategori unit

<Steps>

### Klik kode pada data yang akan diubah, yang tampil di tabel.

`Catatan: Jika ada pesan "Data kategori unit tidak bisa dirubah / dihapus karena sudah digunakan" maka data tersebut tidak bisa diubah maupun dihapus.`

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Lakukan perubahan yang diinginkan.

### Klik tombol **Update** untuk mengedit data.

</Steps>

## Langkah-langkah menghapus kategori unit

<Steps>

### Klik kode pada data yang akan dihapus, yang tampil di tabel.

`Catatan: Jika ada pesan "Data kategori unit tidak bisa dirubah / dihapus karena sudah digunakan" maka data tersebut tidak bisa diubah maupun dihapus.`

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Klik tombol **Hapus** untuk menghapus data.

</Steps>
