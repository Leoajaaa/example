# Tipe Unit

Menu **Tipe Unit** digunakan untuk mengelompokkan Unit Properti
(contoh : tipe 21 dan tipe 36 untuk rumah atau 2BR, 3BR dan Studio untuk Apartemen).

## Langkah-langkah menambah tipe unit

<Steps>

### Klik button **Buat Baru**.

### Isi pada Text Box kode tipe dan nama Tipe.

### Pilih Tipe Unit Berdasarkan Proyek yang sudah disediakan.

![tambah tipe unit 1](@/public/image/web/data-master/properti/tipe-unit/tipe_unit1.png)

### Klik tombol **Simpan**, untuk menyimpan data.

`Catatan: kode dan nama tipe unit wajib diisi.`

</Steps>

## Merubah tipe unit melalui tabel

Anda bisa merubah data tipe unit di tabel dengan mengklik di kolom checkbox pada kolom proyek yang tersedia.

![tambah tipe unit 2](@/public/image/web/data-master/properti/tipe-unit/tipe_unit2.png)

## Langkah-langkah mengubah tipe unit

<Steps>

### Klik kode pada data yang akan diubah, yang tampil di tabel.

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Lakukan perubahan yang diinginkan.

### Klik tombol **Update** untuk mengedit data.

</Steps>

## Langkah-langkah menghapus tipe unit

<Steps>

### Klik kode pada data yang akan dihapus, yang tampil di tabel.

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Klik tombol **Delete** untuk menghapus data.

`Catatan: Jika ada pesan "Maaf, data tipe sudah terdaftar pada item pekerjaan unit" maka tipe unit tersebut tidak bisa dihapus `

</Steps>
