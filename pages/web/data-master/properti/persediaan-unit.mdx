# Persediaan Unit

Menu **Persediaan Unit** digunakan untuk menyimpan data unit properti
pada setiap proyek.

## Cara menampilkan data

<Steps>

### Pilih Proyek.

### Klik tombol “Tampilkan”.

Menampilkan data sesuai kriteria data (Kategori, Tipe, Cluster, No Unit, LB/NETT, LT/SG, View unit dan status) yang diisi
![tampil persediaan unit 1](@/public/image/web/data-master/properti/persediaan-unit/persediaan_unit1.png)

</Steps>

## Langkah-langkah menambah unit

<Steps>

### Klik button **Buat Baru**.

### Pilih proyek, kategori, tipe dan cluster pada masing-masing combo.

### Isi no.unit, LB, LT, view dan spesifikasi bangunan pada masing-masing Text Box.

### Klik tombol **Proses**, untuk menyimpan data.

`Catatan: data proyek, kategori, type, cluster, no. unit, LB dan LT wajib diisi.`

</Steps>

## Langkah-langkah mengubah unit

<Steps>

### Klik no. unit pada data yang akan diubah, yang tampil di tabel.

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Klik tombol **Update** untuk mengedit data.

</Steps>

## Langkah-langkah menghapus unit

<Steps>

### Klik no. unit pada data yang akan dihapus, yang tampil di tabel.

### Sistem akan menampilkan data tersebut, pada masing-masing Text Box.

### Klik tombol **Hapus** untuk menghapus data.

Klik “Oke” untuk menghapus data, Klik “Cancel” untuk membatalkan
penghapusan data.

`Catatan: Jika ada pesan "Data tidak bisa dihapus karena sudah digunakan pada Modul lain" maka data tersebut tidak bisa dihapus.`

</Steps>

## Langkah-langkah mengenerate persediaan unit

<Steps>

### Klik button **Generate Persediaan**.

### Pilih proyek, kategori, tipe dan cluster pada masing-masing combo.

### Isi no.unit, LB, LT, view, dan spesifikasi bangunan pada masing-masing Text Box.

### Isi karakter awalan, jumlah digit, range penomoran awal mulai dari dan nomor akhir pada masing-masing Text Box.

### Klik tombol **Proses**, untuk menyimpan data.

`Catatan: data proyek, kategori, type, cluster, no. unit, LB, LT, karakter awalan, jumlah digit, range penomoran awal mulai dari dan nomor akhir wajib diisi.`

</Steps>

## Langkah-langkah menghapus unit melalui tabel

Pilih unit yang dipilih di kolom "PILIH" pada tabel lalu klik tombol **Hapus** pada menu bar yang tersedia.
