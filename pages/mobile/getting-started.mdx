## Login Apps

Anda bisa memulai aplikasi porta pro mobile dengan login terlebih dahulu:

1. Jalankan aplikasi porta pro mobile pada perangkat anda, lalu akan tampil halaman login seperti berikut:

<Image
    loading="lazy"
    src={screenLogin}
    width={250}
    height={250}
    style={{ margin: "auto" }}
></Image>

2. Isi kode perusahaan anda yang sudah terdaftar pada aplikasi Porta Pro
3. Isi username dan password dengan data user yang telah terdaftar pada aplikasi Porta Pro/ECes.Q

## Lupa Password
Anda dapat meng-klik link **Lupa Password** pada halaman login aplikasi untuk dapat me reset password login anda.
Selanjutnya anda akan diarahkan pada isian halaman lupa password:

1. Isi kode perusahaan anda yang telah terdaftar pada aplikasi
2. Isi alamat email anda
3. Klik/Tap tombol **RESET**
4. Aplikasi akan mengirimkan arahan selanjutnya melalui email yang telah anda isi

<Callout type="info" emoji="ℹ️">
    `Untuk dapat melakukan proses reset password, aplikasi membutuhkan alamat email anda yang telah terdaftar pada aplikasi.
    jadi, pastikan sebelumnya anda sudah melengkapi alamat email pada aplikasi`
</Callout>
