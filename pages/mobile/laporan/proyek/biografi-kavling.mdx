# Biografi Kavling

Modul **Biografi Kavling**, berfungsi untuk memonitor suatu proyek.
Pada modul ini, pengguna dapat memonitor data terkait :

-   Status unit, Progres Bangunan, Progres Legalitas, Cara Bayar
-   Progres Pembayaran, Total Pembayaran, nilai yang sudah dibayar dan sisa pembayaran, langsung dari aplikasi
