# Resume Proyek

Menu **Resume Proyek**, berfungsi sebagai laporan pencatatan perhitungan antara untung dan ruginya pihak developer dalam merencanakan suata anggaran biaya.
Pada modul ini, pengguna dapat melakukan review data terkait :

-   Menampilkan nilai Target Penjualan, KPR, Penerimaan Booking Fee, Target & Cash In, Realisasi Pengeluaran, Progres Bangunan dan Progres Perijinan.
-   Mendownload laporan resume proyek, langsung dari aplikasi
