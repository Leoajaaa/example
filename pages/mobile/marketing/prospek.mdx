#  <span style={{ fontSize: '28px' }}> Prospek Konsumen </span>

Menu **Prospek Konsumen**, berfungsi untuk memfasilitasi pencatatan Prospek yang dilakukan oleh Marketing. Prospek dapat berfungsi untuk mem-blok unit oleh Marketing, yang diperuntukan Calon Customer bila data nomor unit dimasukan pada proses Input data Reserve.

---
<Steps>
##  <span style={{ fontSize: '24px' }}> Preview Prospek Konsumen </span>

1. Anda membuka aplikasi Porta Pro yang telah terinstal di smartphone Anda. Dengan mengklik ikon aplikasi, layar ponsel Anda terangkat dengan tampilan antarmuka aplikasi Porta Pro.

2. jika pada langkah sebelumnya Anda belum melakukan proses login, di halaman awal aplikasi Anda akan menemukan opsi untuk login. Anda dapat memasukkan kredensial akun Anda, seperti nama pengguna dan kata sandi, untuk mengakses akun Anda. Setelah login berhasil, Anda siap untuk melanjutkan.

3. begitu Anda masuk ke dalam aplikasi, Anda akan melihat beberapa modul yang tersedia. Di antara modul-modul tersebut, Anda perlu memilih modul yang berjudul "Marketing". Anda bisa melakukan ini dengan mengetuk atau mengklik opsi "Marketing" yang tertera pada layar Anda.
 
 <Image

## <span style={{ fontSize: '24px' }}>**Langkah-langkah menampilkan data Prospek Konsumen**</span>

1. Buka Aplikasi Porta Pro di Smartphone

2. Setelah itu, jika user belum login maka silahkan login terlebih dahulu.

3. Pilih modul Marketing

4. Pilih menu Prospek Konsumen

5. Pilih Proyek

6. Pilih Status Prospek

<Steps>
## <span style={{ fontSize: '24px' }}> **Tambah Prospek Baru** </span>

### <span style={{ fontSize: '22px' }}>  Pilih tombol buat baru </span>

### Lengkapi Data Prospek pada Halaman Tambah Data

 <Image
            src={prospeka22}
            
            width={250}
            height={250}
                
        />

1. Pilih nama **Proyek**
2. Pilih **Tanggal Prospek, Sumber Info, Status Prospek** 
3. Isi media kontak
4. Isi catatan 
5. Isikan data personal calon prospek pada text box **Nama, No.Handphone, No.Telp, Email, NPWP**
6. Pilih jenis kelamin
7. Isi data alamat pada text box **Provinsi, Kota, dan isikan alamat domisili**  
8. Pilih pekerjaan
9. Isikan pada text box **deskripsi pekerjaan, penghasilan tetap/gaji, dan referral** 
10. Isi Marketing
11. Klik icon upload identitas 
12. Klik icon simpan data

<Callout type="info" emoji="ℹ️">
    `Tanda (*) pada atribut isian, menandakan bahwa isian tersebut wajib diisi/tidak boleh kosong`
</Callout>

</Steps>
