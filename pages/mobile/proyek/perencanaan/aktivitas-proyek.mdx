# <span style={{ fontSize: '28px'}}>Aktifitas Proyek</span>

Modul **Aktifitas Proyek**, berfungsi sebagai pencatatan presentase Aktifitas Proyek sesuai dengan keadaan sebenarnya.
Pada modul ini, pengguna dapat melakukan update data terkait :

-   Nilai presentasi progres pembangunan, berdasarkan bobot pekerjaan yang telah ditentukan sebelumnya
-   Upload photo bukti progres Aktifitas Proyek, melalui aplikasi Porta Pro
-   Bukti photo yang diupload, akan langsung terdapat watermark lokasi saat photo tersebut diambil
