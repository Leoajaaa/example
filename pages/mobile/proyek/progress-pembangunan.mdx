#  <span style={{ fontSize: '28px' }}>Progres Pembangunan Unit</span>

<span style={{ fontSize: '18px' }}> Modul **Progres Pembangunan Unit**, berfungsi sebagai pencatatan presentase progres pembangunan unit sesuai dengan keadaan sebenarnya.
Pada modul ini, pengguna dapat melakukan input data terkait :</span>

-   Nilai presentasi progres pembangunan, berdasarkan bobot pekerjaan yang telah ditentukan sebelumnya
-   Upload photo bukti progres fisik unit, langsung dari aplikasi
-   Bukti photo yang diupload, akan langsung terdapat watermark lokasi saat photo tersebut diambil
